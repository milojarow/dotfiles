#   keyword<TAB>emoji<SPACE>emoji<SPACE>...
#
# Notes
#   - One keyword per line, separated from the emoji list by a TAB.
#   - Comments start with #. Blank lines are ignored.
#   - The same emoji can appear in multiple clusters; all keywords merge.
#   - Add your own clusters anywhere — castellano, jergas, abreviaciones, lo que sea.
#   - Read by ~/.config/sway/scripts/emoji-modi-handler on every picker launch.
