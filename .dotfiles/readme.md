# Milo's Dotfiles

This repository contains my personal configuration files (dotfiles), scripts, and settings for my Arch Linux setup. It includes configurations for various tools and applications that I use regularly, adhering to the KISS philosophy.

## Contents

- Dotfiles for bash, vim, leftwm, etc.
- Scripts for various tasks.
- Xorg configuration files.
- Configuration files located in $HOME/.config.

## Usage

To use these dotfiles, clone the repository and manually link the files to the appropriate locations in your home directory.

**Note:** Please be aware that these configurations are tailored to my personal preferences and setup. They may not work perfectly on your system without modifications.
