# sway-inputs

Input device configuration for sway: keyboards, mice, touchpads, tablets, and seats.

## Sources
- sway-input.5.scd — official input configuration man page

## Files
- SKILL.md — device selectors, config pattern, quick ref
- KEYBOARD_XKB.md — keyboard layout (XKB), repeat rate, numlock
- POINTER_LIBINPUT.md — mouse/touchpad acceleration, tap, scroll, DWT
- ADVANCED_SEAT.md — tablet mapping, seat config, cursor theme, shortcut inhibition
